# TRAEWORK 发布指南

## 方案一：使用 GitHub Releases（最简单，免费）

### 步骤 1：创建 GitHub 仓库

1. 访问 https://github.com
2. 点击右上角的 "+" → "New repository"
3. 仓库名称：`TRAEWORK`（或你喜欢的名字）
4. 选择 Public 或 Private（推荐 Public）
5. 点击 "Create repository"

### 步骤 2：上传代码到 GitHub

在项目目录下打开终端，执行：

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/你的用户名/TRAEWORK.git
git push -u origin main
```

### 步骤 3：创建 Release

1. 在 GitHub 仓库页面，点击右侧的 "Releases"
2. 点击 "Draft a new release"
3. 填写：
   - Tag version: `v1.0.0`
   - Release title: `TRAEWORK v1.0.0`
   - Description: 写一些更新说明
4. 点击 "Attach binaries by dropping them here or selecting them."
5. 上传 `dist/TRAEWORK.exe`
6. 点击 "Publish release"

### 步骤 4：创建 version.json

我已经为你创建了 `version.json` 文件，你需要：

1. 修改 `version.json` 中的 `download_url` 为你的实际下载地址
2. 把 `version.json` 也上传到 GitHub 仓库
3. 或者使用 GitHub Pages 来托管它

### 步骤 5：使用 GitHub Pages 托管 version.json（可选但推荐）

1. 在 GitHub 仓库，点击 "Settings"
2. 左侧菜单找到 "Pages"
3. Source 选择 `main` 分支，文件夹选 `/ (root)`
4. 点击 "Save"
5. 等待几分钟，你的 `version.json` 就可以通过以下地址访问：
   `https://你的用户名.github.io/TRAEWORK/version.json`

### 步骤 6：更新应用配置

修改 `config.json` 或 `traework/core/config.py` 中的 `update_url`：

```json
{
  "app": {
    "update_url": "https://你的用户名.github.io/TRAEWORK/version.json"
  }
}
```

## 方案二：使用 Gitee（码云）- 国内更快

如果你在中国，推荐使用 Gitee，访问速度更快：

1. 访问 https://gitee.com
2. 创建仓库的步骤和 GitHub 类似
3. Gitee Pages 可以用来托管 `version.json`
4. Gitee Releases 可以用来上传 .exe 文件

---

## 方案三：本地测试服务器（开发时用）

如果你只是想在本地测试更新功能，可以用 Python 启动一个简单的 HTTP 服务器：

```bash
# 在项目目录下运行
python -m http.server 8000
```

然后配置 `update_url` 为：
```
http://127.0.0.1:8000/version.json
```

---

## 发布新版本的流程

当你要发布 1.1.0 版本时：

1. 修改 `traework/core/config.py` 中的版本号为 `1.1.0`
2. 重新打包：`pyinstaller traework.spec`
3. 在 GitHub/Gitee 创建新的 Release，tag 为 `v1.1.0`
4. 上传新的 `TRAEWORK.exe`
5. 更新 `version.json` 中的版本号和下载地址
6. 提交并推送 `version.json` 到仓库

---

## 注意事项

⚠️ **重要**：
- 确保 `version.json` 可以被公开访问
- 下载地址必须是直接下载链接，不能是 HTML 页面
- 版本号建议使用语义化版本：`主版本.次版本.修订号`（如 1.0.0, 1.1.0, 2.0.0）
