# 🚀 TRAEWORK 超级简化发布指南

## 只需要 3 步！

---

## 第一步：第一次配置（只需做一次）

### 1. 创建 GitHub 仓库
- 访问 https://github.com
- 点击右上角 + → New repository
- 仓库名：`TRAEWORK`
- 点击 Create repository

### 2. 上传代码
在项目文件夹打开终端，依次输入：
```bash
git init
git add .
git commit -m "First commit"
git branch -M main
git remote add origin https://github.com/你的用户名/TRAEWORK.git
git push -u origin main
```

### 3. 开启 GitHub Pages
- 在 GitHub 仓库页面，点击 Settings
- 左侧点击 Pages
- Source 选 `main` 分支，文件夹选 `/ (root)`
- 点击 Save

### 4. 修改配置
打开 `release.py`，找到第 72 行，把：
```python
"download_url": "https://github.com/你的用户名/TRAEWORK/releases/download/v{}/TRAEWORK.exe"
```
改成你的实际 GitHub 用户名！

---

## 第二步：以后发布新版本（只需 3 步！）

### 第 1 步：运行发布脚本
在终端输入：
```bash
python release.py
```
然后按提示输入：
- 新版本号（比如 1.1.0）
- 更新说明（可选）

脚本会自动：
- ✅ 更新版本号
- ✅ 更新 version.json
- ✅ 打包成 .exe

### 第 2 步：提交到 GitHub
```bash
git add .
git commit -m "Release v1.1.0"
git push
```

### 第 3 步：在 GitHub 上传 .exe
- 在 GitHub 仓库点击 Releases
- 点击 Draft a new release
- Tag 填：`v1.1.0`
- 上传 `dist/TRAEWORK.exe`
- 点击 Publish release

---

## 完成！🎉

就这么简单！以后别人从 GitHub 下载你的软件，打开就能自动检查更新了！

---

## 💡 常见问题

**Q: 别人下载后能检查更新吗？**
A: 可以！只要你配置好了 update_url！

**Q: update_url 是什么？**
A: 就是 `https://你的用户名.github.io/TRAEWORK/version.json`

**Q: 我需要改代码里的 update_url 吗？**
A: 要的！在 `config.json` 和 `traework/core/config.py` 里改成你的 GitHub Pages 地址！
