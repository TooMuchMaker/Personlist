# TRAEWORK 计划管理系统使用指南

## 系统简介

这是一个支持用户和TRAE共同使用的智能计划管理系统，支持长期、中期、短期三类计划，并可以自动提取当前阶段应该执行的计划。

---

## 快速启动

### 方式一：Web可视化界面（推荐）

双击 `启动计划管理.vbs` 即可：
- 自动启动Web服务（无窗口）
- 自动打开浏览器访问 `http://127.0.0.1:5001`

### 方式二：命令行界面

```bash
cd "Maker的文档/计划管理"
python plan_manager.py
```

---

## Web界面功能

### 1. 当前阶段设置
- 设置阶段名称和时间范围
- 自动提取该阶段应该执行的计划

### 2. 添加新计划
- 选择计划类型（长期/中期/短期）
- 填写计划标题和描述
- 设置优先级和日期
- 添加标签
- 选择创建者（用户/TRAE）

### 3. 查看计划
- **阶段计划**：自动提取当前阶段的计划
- **短期计划**：1-3个月的计划
- **中期计划**：3-6个月的计划
- **长期计划**：6-12个月的计划

### 4. 排序和筛选
- 按优先级排序
- 按开始日期排序
- 按结束日期排序
- 按创建时间排序
- 按状态筛选

### 5. 编辑和删除
- 点击编辑按钮修改计划
- 点击删除按钮删除计划
- 更新进度条

---

## 功能特性

### 三类计划支持
- **长期计划 (long_term)**：6-12个月的目标和规划
- **中期计划 (mid_term)**：3-6个月的目标和规划
- **短期计划 (short_term)**：1-3个月的目标和规划

### 阶段计划
- 设置当前阶段的时间范围
- 自动提取三类计划中在当前阶段应该执行的计划
- 支持多种排序方式

### 状态管理
- pending (待处理) ⏳
- in_progress (进行中) 🔄
- completed (已完成) ✅
- cancelled (已取消) ❌

### 优先级设置
- high (高优先级) 🔴
- medium (中优先级) 🟡
- low (低优先级) 🟢

## TRAE使用方法

作为AI助手，我可以通过以下方式帮助您管理计划：

```python
import sys
sys.path.append('Maker的文档/计划管理')
from plan_manager import PlanManager

manager = PlanManager('Maker的文档/计划管理/plans.json')

# 添加计划
manager.add_plan(
    plan_type="short_term",
    title="完善文件管理器功能",
    priority="high",
    start_date="2026-02-01",
    end_date="2026-04-30",
    creator="trae"
)

# 查看计划
manager.list_plans(sort_by="priority")

# 设置当前阶段
manager.set_current_stage(
    name="基础设施完善阶段",
    start_date="2026-02-01",
    end_date="2026-04-30"
)

# 查看阶段计划
manager.get_stage_plans(sort_by="priority")
```

## 注意事项

1. Web服务运行在端口5001
2. 所有日期格式为 YYYY-MM-DD
3. 进度为0-100的整数
4. 创建者可以是 "user" 或 "trae"
5. 定期保存和备份数据
